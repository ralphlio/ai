# AI & Vibe Coding Lab

## 中文

这个仓库是我用来体验 AI 与 vibe coding 各种能力的实验场。这里会持续收集由 AI 协助完成的网页、交互工具、小游戏、知识文章和产品原型，用来观察 AI 在内容创作、前端设计、代码生成、交互实现、文档组织和快速发布中的实际表现。

项目不追求单一主题，更像一个不断生长的作品目录：有些页面用于解释 AI、Agent、MCP、Skill 等概念，有些页面用于测试网页视觉与交互，有些页面则是小游戏或实用工具。每一次更新都是一次小实验。

## English

This repository is my playground for exploring AI and vibe coding capabilities. It collects web pages, interactive tools, mini games, knowledge articles, and small prototypes created with AI assistance, so I can observe how AI performs across content creation, frontend design, code generation, interaction design, documentation, and fast publishing.

The project is not limited to one topic. It is a growing directory of experiments: some pages explain AI, agents, MCP, and skills; some test visual design and browser interactions; others are games or practical tools. Each update is a small experiment in building with AI.

Live site:

[https://ralphlio.github.io/ai/](https://ralphlio.github.io/ai/)

Current pages:

- `index.html`: AI knowledge directory
- `codex-architecture.html`: Codex architecture and agent product design
- `agent-mcp-skill.html`: Concepts and relationships between Agent, MCP, and Skill
- `pixel-maze.html`: Pixel maze mini game
- `sankey-tool.html`: Sankey diagram generator
- `github-guide.html`: A beginner-friendly GitHub guide

## Notes

This repo is hosted with GitHub Pages. Most pages are standalone HTML files, which keeps the project easy to inspect, edit, and publish.
